🧠 三层负荷优化 (AI 时代减负方案)

基于认知科学理论，降低 AI 协作中的切换代价、上下文压力和焦虑负荷， 让人类从"AI 管理者"回归"决策者和创造者"

Phase 1

进度连续性记忆

减少任务切换的认知恢复成本

  • 记住上次查看位置
  • Session 阅读进度追踪
  • 智能推荐下一个 Session
Phase 2

AI 上下文摘要

突破 7±2 工作记忆上限

  • 3 句话概括 Session 状态
  • 批量摘要用于 Dashboard
  • Task 简短摘要
Phase 3

置信度指标

消除 AI 焦虑负荷

  • AI 输出置信度评分
  • 三级动作建议
  • 变更历史追踪
95%+ 自动执行 70-95% 通知 <70% 确认
Phase 4

SOP + 决策边界

AI 定位从伙伴变工具

  • 可定义标准操作流程
  • 人类决策边界明确化
  • 高风险操作强制确认

🏗️ 系统架构

OpenCode Task Hub
├── Gateway Server (HTTP + WebSocket)
├── Session Manager (树形结构 + Checkpoint)
├── Task Manager (增强看板)
│   ├── AI Enhancer (智能分析)
│   ├── Webhook Manager (外部集成)
│   └── Analytics Engine (数据报表)
├── Database (SQLite/PostgreSQL)
└── Channel Plugins (CLI / Web / API)

🚀 快速开始

# 克隆项目
git clone https://github.com/lxztry/opencode-task-hub.git
cd opencode-task-hub

# 安装依赖
npm install

# 启动服务
npm start

# 访问Dashboard
http://localhost:3030

# 创建会话
curl -X POST http://localhost:3030/api/enhanced/sessions \
  -H "Content-Type: application/json" \
  -d '{"name":"Sprint 1","type":"sprint"}'

# AI 任务分析
curl http://localhost:3030/api/ai/analyze/task_123
