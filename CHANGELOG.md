# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-22

### Added

- Real-time task hub server with WebSocket support
- Dashboard UI for monitoring OpenCode sessions
- Task management (create, update, delete, assign)
- Session activity tracking with heartbeat system
- Cross-platform installer scripts (install.sh, install.bat)
- OpenCode plugin for automatic session registration
- REST API for all operations
- WebSocket events for real-time updates
- Persistent data storage (data.json)
- npm package for easy installation
- Comprehensive documentation (README, CONTRIBUTING)
- GitHub repository integration
- CI/CD with GitHub Actions
- Unit tests with Node.js test runner
- Code linting scripts
- SVG logo
- Additional plugins (progress-tracker, metrics-collector)

### Features

- Multi-session monitoring across projects
- Real-time activity updates
- Task creation and assignment
- Session heartbeat (30s interval)
- Cross-platform support (macOS, Linux, Windows)
- Docker support
- npm release automation
- Docker Hub automated builds

## [0.1.0] - 2026-03-21

### Added

- Initial project setup
- Basic Express server
- Session registration API
- Task management API
- WebSocket server
- Basic dashboard UI
