# OpenCode Task Hub

管理多个项目任务，实时同步到仪表板。

## 命令

### 创建并报告任务
```
我正在开始 [任务描述]，请使用 task-hub 上报任务
```
任务将自动创建并标记为进行中。

### 更新任务进度
```
任务进度已更新到 [百分比]
```
自动更新当前任务进度。

### 标记任务完成
```
任务已完成，请更新 task-hub 状态
```
将任务标记为已完成 (100%)。

### 查询任务状态
```
查看当前项目有哪些任务
```
从仪表板获取任务列表。

## 内部工作

- 通过 HTTP API 与本地任务服务器通信
- 服务器地址: http://localhost:3030
- 自动在每个 opencode 会话中启用任务追踪
